Vulnerability Assessment

Turn a software inventory into a decision-ready vulnerability report — the Vulners-style audit. Paste OS packages (dpkg -l, rpm -qa), language packages (pip / npm / PURL), CPE URIs, or plain product version lines. Each component is matched against XORCISM's enriched vulnerability store and scored with CVSS, EPSS, CISA KEV, SSVC and Exploit-DB availability into Act / Prioritise / Track. Runs entirely on your data.
