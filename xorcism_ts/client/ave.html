AVE — Agentic Vulnerability Enumeration

A behavioral-classification standard for agentic-AI components (AI agents, skill files, MCP servers, system prompts). Where CVE/OSV need a package version, AVE assigns stable AVE-2026-##### ids and AIVSS scores (OWASP AIVSS: CVSS base × agentic amplification) to behavioral vulnerability classes, mapped to the OWASP Agentic Top 10, OWASP MCP Top 10, MITRE ATLAS and NIST AI RMF. From bawbel/ave (Apache-2.0).
